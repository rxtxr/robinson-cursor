Willie

WIP Restoration of Steamboat Willie (1928) — first 60 s.

Flat-field → deflicker (v6) → nlmeans denoise → mild unsharp.

Left: source scan. Right: restored pipeline output.

What works · what doesn't

  • flicker — dark & mid greys: visibly reduced
  • flicker — bright greys: still insufficient
  • shadow / vignette: flat-field fix landed, more tuning possible
  • stabilization: failed — vidstab fights hand-drawn motion
  • noise reduction: unsatisfying — softens fine line strokes
  • dust / scratch removal: failed — eats 1–2-frame animation strokes
  • upscale: Real-ESRGAN pipeline scripted, not yet run end-to-end

Planned next steps

  • Content-masked percentile gain to catch bright-grey flicker
  • Real-ESRGAN 4× upscale in 100-frame batches (not in /tmp)
  • Post-upscale unsharp
  • Optional RIFE 24 → 48 fps
  • Audio hiss reduction on the original Opus track
  • Proper before / after scrubber viewer

Note

Archival film restoration is hard, and animation breaks most of the off-the-shelf tools built for live action (temporal medians eat line strokes, vidstab fights drawn motion, radial vignette filters don't match asymmetric gate shading). This day ships the best 1-minute pass of the current pipeline; the project will be picked up again later.
