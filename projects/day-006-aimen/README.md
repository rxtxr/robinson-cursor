# Day 006 — AiMEN BREAK

Browser-based breakbeat chopper and step sequencer. Slice, arrange, add effects, layer a bass synth on top.

## Features

- 5 included breaks, drag & drop WAV upload, BPM detection
- Slice bank with automatic drum classification (kick, snare, hat, ghost)
- Step sequencer with per-step pitch, gain, offset, retrigger, probability, reverse, tape stop
- FX rack: filter + LFO, flanger, delay, reverb, drive, bitcrusher, vinyl emulation, swing
- Bass synth: 808 Sub, Reese, Hoover, 303 Saw with glide, sub layer, detune LFO
- Undo/redo, keyboard shortcuts, WAV export, pattern save/load

## Stack

Pure Web Audio API, Canvas 2D, vanilla JS. No frameworks, no build step.
